# Hugging-Face exercise : Model Search Engine

You need node 12+ to use the app :

## Install

```
npm install
```

## Test the app 

```
npm run test:all
```

You can run in watch mode individually :
Unit tests :

```
npm run test:unit
```

Integration tests :

```
npm run test:integration
```

End to end tests :

```
npm run test:e2e
```

## Start the app 

### To start with IN_MEMORY database :

```
npm start
```

### To start with JSON database :

```
npm run start:json
```
